*General-purpose robots adaptable for clinical and research applications*

**Target Count**: 15+ models
**Status**: Development

## Overview

This category contains general-purpose collaborative and industrial robots that can be adapted for oncology clinical trial applications. These robots serve as platforms for research, development, and non-certified clinical assistance tasks.
